# Proyecto_c
## Descripcion
En este repositorio se encontraran un conversor de monedas en lenguaje c 
## Prerequisitos 
1. Tener una IDE con la que trabajaremos
2. Tener sistema linux o una pc virtual(si no se cuenta con windows)
3. Tener los paquetes de instalacion
    - Lenguaje en c
    - git

## Pasos para la ejecucion
1. Abrimos nuestra IDE
2. En la terminal verificamos si los paquetes tienen una actualizacion (**sudo apt update**) 
3. Instalamos c con el siguente comando en la terminal **sudo apt install build-essential**
4. Creamos un archivo nuevo nombre.c
5. Copiamos el codigo del archivo en el repositorio y lo pegamos en el que creamos
6. Copilamos el archivo para su visualizacion con el comando **gcc nombredelarchivo.c -o conversor** (<- este crea un archivo con el nombre "conversor")
7. En la terminal ejecutamos el programa con el comando **./conversor**
